# Writing Quest

Retro writing desktop app for indie writers and hobbyists.

## Status

Repo: `tbelskie/writing-quest`

Current read: healthy skeleton with Tauri, React, Rust, SQLite, quests, sprites, project creation, editor, autosave/persistence, and export hooks. Needs launch-readiness cleanup before it can be sold confidently.

## Current Bet

Turn Writing Quest into the portfolio's first polished paid launch candidate.

## Launch Lessons

- paid desktop app launch
- seasonal writing challenge timing
- one-time license positioning
- local-first trust
- UX writing for creative flow
- export and persistence reliability

## Known Risks

- lingering naming drift from Cartridge to Writing Quest
- export implementation may not produce production-valid DOCX/EPUB
- license verification path is placeholder-level
- assets and onboarding need polish
- launch window requires discipline

## Next Actions

1. Run a launch-readiness audit.
2. Create a v1 scope.
3. Fix naming and trust issues.
4. Decide pricing and distribution.
5. Build docs/site/store surface.
