# v0.17 Clear First Command Queue Plan

v0.16 made Portfolio the default command center. v0.17 makes that command center more opinionated.

The next product move is:

> Pendragon should tell the founder what to clear first across the portfolio.

## Goal

Add a ranked Clear First queue to Portfolio that turns product-room signals into concrete unblocking work.

The founder should not have to inspect every tab to decide what matters. Pendragon should synthesize the room and route them to the right module.

## Inputs

Each product room contributes Clear First candidates from:

- stage mismatch
- readiness gate blockers
- critical docs still open
- launch surface QA blockers
- missing proof links
- unresolved decisions
- missing setup/stage assets
- readiness gate risks
- next warroom action fallback

## Ranking

Each candidate gets a base rank by severity. The portfolio queue then adds a fraction of the product attention score so high-risk products rise without hiding urgent blockers elsewhere.

The queue is intentionally practical, not scientific. It should answer:

> What should I unblock before adding more work?

## Routing

Every queue item routes directly to the module where the founder can act:

- Setup for stage mismatch and setup gaps
- Gate for readiness blockers and risks
- Docs for critical docs work
- Forge for launch surface QA blockers
- Evidence for proof gaps
- Decisions for unresolved choices
- Warroom for general momentum

## Non-Goals

Do not add:

- task assignment
- due dates
- recurring notifications
- external integrations
- drag-and-drop priority editing
- automatic issue creation

Those are later platform behaviors. v0.17 is about decisive local synthesis.

## Acceptance Criteria

- Portfolio shows a Clear First section beneath portfolio metrics.
- Clear First contains ranked queue items across products.
- Each item shows rank, severity, source module, product, title, detail, and product attention score.
- Each item has a button that opens the right product and module.
- Portfolio metrics include total detected Clear First moves.
- Product cards surface their top Clear First item.
- The app asset version bumps to `0.17.0`.
- Existing rooms and Forge continue to render.
