## Interaction Notes

- Primary action is `Download HTML draft`.
- The preview is read-only in v0.6.
- Editing happens in Brief, Docs, Decisions, and Gate.
- Forge should reveal missing truth instead of hiding gaps with slick copy.

## Visual Direction

- Keep the Lava Glass palette.
- Use a premium dark launch-page preview rather than a busy dashboard.
- Give the generated page its own framed preview surface.
- Avoid a dense grid; this module should feel like a page being forged from the room.

## Founder Value

The user should feel:

> I can see the thing I am trying to launch.

That is the product moment. The rest of Pendragon exists to improve this output.
