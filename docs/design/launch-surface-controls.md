# Launch Surface Controls

Launch Surface Controls are the smallest editable layer between the product room and the generated page.

## Layout

```text
Forge / Paid beta page

[Download HTML draft] [Edit Brief] [Open Docs]

Readiness signals
Surface strategy

Launch Surface Controls
Make the generated page publishable.        [Save controls]

[Offer]
[CTA label] [CTA URL]
[Support URL] [Trust claim]
[Founder note]

Generated launch surface
---------------------------------------------------------
Beta offer
$19 early access...

Primary CTA
Join paid beta
Open CTA link, if present

Trust Claim
Writing Quest is offline-first...

Founder Note
Use this as the paid beta page skeleton...
---------------------------------------------------------
```

## Rules

- Keep controls plain text.
- CTA URL and support URL can be blank.
- If a URL is present, it must be `http` or `https`.
- Controls should sharpen the generated surface, not become a full CMS.

## Why This Exists

Forge needs two kinds of truth:

- inferred truth from the product room
- founder-owned publishing details

This is the smallest useful bridge between the two.
