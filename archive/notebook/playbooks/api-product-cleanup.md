# API Product Cleanup

Use this for MockBizOps and future developer products.

## Checks

- local setup works
- public demo works
- docs match actual routes
- OpenAPI spec is reachable
- seed data is repeatable
- examples are accurate
- auth story is clear
- rate limits/pricing are honest
- tests can run without production infrastructure

## Output

- broken setup list
- docs drift list
- demo readiness verdict
- minimum launch checklist
