# Chrome Extension Growth Check

Use this for Dot Primer and any future browser extension product.

## Check

- store impressions
- store page views
- installs
- uninstalls
- enabled vs disabled
- weekly users
- daily users
- reviews/ratings
- search terms if available
- listing screenshots
- listing headline
- onboarding flow
- permissions copy

## Questions

- Are people finding it?
- Are store visitors installing it?
- Are installed users keeping it enabled?
- What is the retained use case?
- What would make the product worth paying for?
